# 📊 ScaleSale Analytics Platform

Upload sales data, get instant insights. Built with Python & Streamlit.

## 🚀 Live Demo
[Try it here](https://your-app.streamlit.app)

## ✨ Features
- Upload CSV/Excel files
- Automatic data cleaning
- Interactive dashboards
- Download reports

## 🛠️ Installation
```bash
pip install -r requirements.txt
streamlit run scalesale_app.py
```
